Documento guida per la creazione di una Virtual Machine su Server

Collegamento fisico del pc al server

1. Collegare il computer allo switch tramite cavo ethernet.

2. In caso di non possedimento di un server DHCP assegnare manualmente un indirizzo IP statico al computer.

3. Collegare lo switch al server tramite un cavo di rete.

4. Accedere al portale VSphere Client attraverso l’IP del server.

Creazione della Virtual Machine su VSphere

1. Accedere tramite le credenziali a VSphere in HTML 5.

2. Andare nella sezione Host and Clusters con l’icona in alto a sinistra.

3. Accedere alla configurazione della VM premendo tasto destro sull’IP nel menu a destra (nel nostro caso 192.168.5.250) e scegliendo l’opzione “New Virtual Machine”.

4. Selezionare una tipologia di creazione (nel nostro caso Create).

5. Rinominare la VM e selezionare una cartella in cui memorizzarla.

6. Selezionare la macchina su cui eseguire la VM.

7. Selezionare la memoria da dedicare alla configurazione e ai file di disco.

8. Selezionare la compatibilità per la virtual machine in base alle capacità del server su cui si hosta la VM.

9. Selezionare un sistema operativo per la VM (nel nostro caso windows server 2016-64bit).

10. Scegliere le risorse hardware del server da dedicare alla VM.

11. Premendo finish creare la virtual machine

Setup della VM come Web Server

1. Aprire il Server Manager installato automaticamente su windows server

2. Premere il bottone Manage > Add Roles and Features

3. Schiacciare Next nella pagina Before You Begin

4. Scegliere Role-based or feature-based installation e premere next

5. Scegliere come server la propria VM e premere next

6. Scegliere Web Server IIS, cliccare Next e selezionare Add Features

7. Come features selezionare quelle necessarie, in questo caso .NET 4.7 che andremmo poi ad aggiornare alla versione 7 più tardi

8. Su web Server Role, premere Next

9. Selezionare come Role Services quelli predefiniti e premere Next

10. Su confirmation premere Install

11. Durante la fase result, aspettare la fine del caricamento e premere Next

Creazione del sito internet tramite IIS

1. Installare il pacchetto ASP.NET hosting bundle 7.0

2. Aprire il programma Internet Information Services (IIS) Manager

3. Aggiungere un nuovo sito premendo tasto destro su Sites e scegliendo l’opzione Add Website

4. Scegliere il nome del sito (nel manager) e il path in cui tenerlo: C:\inetpub\wwwroot.

Pubblicazione del sito

1. Aprire il progetto Visual Studio del proprio sito > tasto destro sul progetto > premere su Pubblica

2. Scegliere come opzione “Cartella” e premere avanti

3. Inserire il path in cui abbiamo creato la cartella per contenere il sito precedentemente, aggiungendo un’ulteriore cartella (in questo caso PctoWebSite)

4. Dare i permessi al software IIS Manager alla cartella in questione

5. A questo punto si può accedere al sito tramite il browser inserendo l’ip della macchina IIS e il nome del file html (192.168.5.14/index.html)
